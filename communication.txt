Communication
=============

== Mailing Lists

The project communicate on the following mailing lists

. https://lists.libremesh.org/mailman/listinfo/lime-dev[lime-dev at lists.libremesh.org] -
  This list is used for general development related work. link:https://www.google.es/search?q=site%3Alists.libremesh.org%2Fpipermail%2Flime-dev[Search archives].
. https://lists.libremesh.org/mailman/listinfo/lime-users[lime-users at lists.libremesh.org] -
  This list is used for project organisational purposes and is **the only official user support channel**. link:https://www.google.es/search?q=site%3Alists.libremesh.org%2Fpipermail%2Flime-users[Search archives].
. https://lists.libremesh.org/mailman/listinfo/lime-es[lime-es at lists.libremesh.org] -
  Lista para recibir soporte e intercambiar información con la comunidad de LibreMesh en español.


== Chatroom

The chatroom can be accessed through link:https://element.io/personal[Element app, desktop software or web client] and entering the +#libremesh-dev:matrix.guifi.net+ chatroom. To write in the chatroom you'll need to register a user on any of the Matrix nodes (the default on element.io will also work). Other options for connecting can be found link:https://matrix.to/#/#libremesh-dev:matrix.guifi.net[clicking here].
