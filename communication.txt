Communication
=============

== Mailing Lists

The project communicate on the following mailing lists

. https://lists.libremesh.org/mailman3/lists/lime-dev.lists.libremesh.org/[lime-dev at lists.libremesh.org] -
  This list is used for general development related work.
. https://lists.libremesh.org/mailman3/lists/lime-users.lists.libremesh.org/[lime-users at lists.libremesh.org] -
  This list is used for project organisational purposes and is **the only official user support channel**.
. https://lists.libremesh.org/mailman3/lists/lime-es.lists.libremesh.org/[lime-es at lists.libremesh.org] -
  Lista para recibir soporte e intercambiar información con la comunidad de LibreMesh en español.


== Chatroom

The chatroom can be accessed through Element app, desktop software or web client (link:https://element.io/personal[click here for checking out these chat clients]) and entering the `#libremesh-dev:matrix.guifi.net` chatroom. To write in the chatroom you'll need to register a user on any of the Matrix nodes (the default on element.io will also work). Other options for connecting can be found link:https://matrix.to/#/#libremesh-dev:matrix.guifi.net[clicking here].
